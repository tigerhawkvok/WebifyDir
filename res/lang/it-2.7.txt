CREARE ARCHIVI

1) E'possibile ggiungere oggetti (file o cartelle) al layout dell'archivio, dal menu contestuale del sistema (tasto destro del mouse), dal menu "Invia a", tramite il drag and drop o dai menu e pulsanti dell'applicazione;
2) Cliccare il pulsante "Crea" per iniziare la creazione dell'archivio.

Usare:
- tab "Opzioni" per modificare il formato, i parametri di compressione e cifratura dell'archivio; 
- tab "I/O" per le opzioni di input e output;
- tab "Filtri" per impostare filtri di esclusione avanzati, se disponibili per il formato prescelto;
- tab "Console" per analizzare / personalizzare / esportare come riga di comando la sintassi del lavoro di archiviazione.

In alternativa, in un singolo passo, è possibile archiviare ciascun oggetto selezionato in un archivio separato usando il menu contestuale di sistema; usare "Invia a" per archiviare più oggetti nello stesso archivio. 

Visualizzando il layout dell'archivio sono possibili anche le seguenti operazioni:
- Doppio click su un oggetto per aprirlo
- Selezionare un file e premere il tasto "\", o "File tools > Dividi file" per dividere il file in volumi delle dimensioni specificate
- Selezionare un file e premere il tasto "<", o "File tools > Compara file" per confrontarlo byte per byte con un altro file
- Selezionare anche più file contemporaneamente e premere il tasto "?", o "File tools > Calcola checksum/hash" per calcolare i valori degli algoritmi di checksum e hash prescelti dei file selezionati
- Selezionare anche più file contemporaneamente e premere il tasto "*" o "Cancellazione sicura" per eliminare il file in maniera non recuperabile


APRIRE ARCHIVI

1) Aprire un archivio, es doppio click su un tipo file associato a PeaZip, o dal menu contestuale o "Invia a" del sistema, o attraverso i menu e pulsanti di PeaZip
2a) Cliccare il pulsante "Estrai tutto", o "Estrai tutto in..." per specificare la destinazione;
2b) Fare doppio click (o premere invio) su un oggetto archiviato per vederne un'anteprima con l'applicazione associata; allo stesso modo se l'oggetto è una cartella sarà aperta, o se è a sua volta un'archivio sarà aperto in una nuova istanza di PeaZip
2c) Selezionare anche più oggetti archiviati contemporaneamente e premere il tasto centrale del mouse o la barra spaziatrice per estrarli (specificare la destinazione);
2d) Fare click con il pulsante destro del mouse per attivare il menu contestuale con le funzioni disponibili, es. estrazione, anteprima, verifica, aggiungi/elimina file ecc
Suggerimento: per estrarre il contenuto in una nuova cartella, usare lo switch "Estrai in nuova cartella" nel menu contestuale o nel tab "I/O", oppure premere il tasto "\" o Ctrl+N

Usare:
- tab "Opzioni" per impostare le opzioni specifiche per il formato di archivio;
- tab "I/O" tab per le opzioni di input e output, come lo switch "Estrai in nuova cartella" (disponibile anche nel menu contestuale di PeaZip);
- tab "Filtri" per impostare filtri avnzati di inclusione ed esclusione, se supportati dal formato dell'archivio;
- tab "Console" per analizzare / personalizzare / esportare come riga di comando la sintassi del lavoro di estrazione.

In alternativa, dal menu contestuale di sistema è possibile estrarre più archivi per volta.


PEAZIP

PeaZip è un programma Open Source per la gestione di file e archivi, con funzioni di cifratura forte; supporta numerosi formati di archiviazione e compressione.

Scrittura: 7Z, 7Z-sfx, ARC, BZ2, GZ, PAQ/LPAQ, PEA, QUAD, split, TAR, UPX, ZIP

Lettura: la lista aggiornata è disponibile su http://peazip.sourceforge.net/peazip-free-archiver.html

E' possibile gestire in lettura e scrittura archivi di formato personalizzato specificando l'eseguibile da utilizzare e la relativa sintassi.

Altre funzioni: dividere/unire file, cancellazione sicura, comparazione, calcolo di checksum e hash, benchmark di sistema, generazione di password e file-chiave casuali.

PeaZip permette di modificare, salvare e ricaricare layout di archivi definiti in precedenti sessioni; applicare potenti filtri multipli ai contenuti dell'archivio; gestire la creazione o estrazione di più archivi per volta; esportare la definizione dei lavori come riga di comando; cifrare con AES256 ecc...
PeaZip_portable è una applicazione portabile con le stesse funzionalità di PeaZip; non richiede installazione, non modifica il sistema su cui gira e può essere usata da unità removibili (memorie USB, percorsi di rete ecc...).
Le funzioni principali del programma sono raggiungibili dal menu contestuale e "Invia a" di Windows e dai menu di Gnome e KDE in Linux (tramite file .desktop); l'integrazione nel sistema è automatica per i pacchetti installabili, ma può essere estesa customizzata sia per i pacchetti installabili che portabili (vedere documentazione online).
PeaZip è cross platform e gira su sistemi a 32 e 64 bit, Windows (9x, 2000, XP, Vista) e Linux.

Download e changelog: http://peazip.sourceforge.net/
Tutti i download: http://sourceforge.net/projects/peazip/
Documentazione online: http://peazip.sourceforge.net/peazip-help.html
FAQ: http://peazip.sourceforge.net/faq.html
