ARCHÍVUMOK LÉTREHOZÁSA

1) Elemek hozzáadása az archívum elrendezéshez a szövegkörnyezeti és Küldés menüből, a húzd és ejtsd vagy az alkalmazás menük és gombok segítségével;
2) Kattints a "Készítés" gombra az archívum létrehozásának indításához.

Használat:
- "Beállítások" fül az archívum formátumok, tömörítési szintek és titkosítási paraméterek testreszabásához; 
- "I/O" fül a be illetve kimeneti beállításokhoz;
- "Szűrők" a fejlett szűrők használatának kizárásához, ha elérhető a választott archívum formátumhoz;
- "Konzol" fül az elemzési és szintaktikai feladatok testreszabásához, vagy a feladatok parancssori vezérléséhez.

Egy menetben, elemeket adhatsz az elkülönített archívumokhoz a rendszer szövegkörnyezeti menüjéből. A Küldés menüből is hozzáadhatsz több elemet a különálló archívumhoz.

In archive layout you can also:
- Dupla kattintás egy elemre a megnyitásához
- Válassz ki egy fájlt és nyomd meg a "\" vagy "Fájl eszközök > Fájl darabolás" gombot a vágáshoz
- Válassz ki egy fájlt és nyomd meg a "<" vagy "Fájl eszközök > Fájlok összehasonlítása" gombot, egy másik fájllal való (bájtról-bájtra) összehasonlításhoz
- Válassz ki több elemet és nyomd meg a "?" vagy "Fájl eszközök > Ellenőrző összeg/hash fájl(ok)" az ellenőrző összeg / hash fájlok létrehozásához
- Válassz ki több elemet és nyomd meg a "*" vagy "Fájlok végleges törlése" gombot a biztonságos fájl törléshez


ARCHÍVUMOK MEGNYITÁSA

1) Open archive(s) from system's context and SendTo menu entries, or by file association, or select one or more archives in PeaZip browser
2a) Click "Extract all" to extract the archive, or "Extract all to" for being asked about output path;
2b) Doubleclick or press enter on an archived object to preview it with associated application, or open it in PeaZip if archive, or explore it if directory;
2c) Select archived objects and press middle mouse button or the spacebar to extract them (being asked for output path);
2d) Rightclick in PeaZip browser to use context menu entries to perform all available functions, like extract selected or displayed objects, preview, test, list, add or delete files etc...
Hint: if you want to extract archive to a fresh new directory, use "Extract in new folder" switch in context menu (or I/O tab), or press \ or Ctrl+N while in the archive browser.

Use:
- "Beállítások" fül a formátum specifikus paraméterek testreszabásához;
- "I/O" fül a be illetve kimeneti beállításokhoz, beleértve a "Kicsomagolás új mappába" opciót is (ez szintén elérhető a szövegkörnyezeti menüből);
- "Filters" tab for using advanced inclusion and exclusion filters, if available for the archive's format;
- "Konzol" fül az elemzési és szintaktikai feladatok testreszabásához, vagy a feladatok parancssori vezérléséhez.

You can extract multiple archives at once from system's context menu or from PeaZip browser.

Archive types unknown to PeaZip can be supported (for creation and extraction) through an appropriate custom executable, selecting "Custom" format in "Option" tab.

A PEAZIP PROGRAMRÓL

A PeaZip, egy ingyenes, hordozható, open source archíváló, titkosító és fájl daraboló eszköz; számtalan archívumot és tömörítési formátumot támogat.

Írás: 7Z, 7Z-sfx, ARC, BZ2, GZ, PAQ/LPAQ, PEA, QUAD, vágás, TAR, UPX, ZIP

Olvasás: a teljes lista itt tekinthető meg - http://peazip.sourceforge.net/peazip-free-archiver.html

Custom type archives can be handled for writing and reading specifying custom executables and syntax in "Option" tab.

További funkciók: fájlok vágása/egyesítése, fájlok törlése (végleges törlés), összehasonlítás, ellenőrző összeg és hash fájlok, rendszer teljesítmény teszt, véletlenszerű jelszavak és kulcsfájlok generálása.

A PeaZip lehetővé teszi a szerkesztés, mentés és archívum szerkezet helyreállítást; apply powerful multiple filters to archive's content; handle multiple archives at once; export job definition as command line; encrypt with AES256 etc.
PeaZip_portable is a standalone application with same features of PeaZip; it doesn't need installation and can be used from removable units (USB keys, remote paths...) not performing any change in the host system.
Most used functions are integrated in SendTo and context menu (on Windows) and as .desktop files for contextual menus (on Linux); system integration is authomatic in installable packages and can be customized by users both for installable and portable packages (tekintsd meg az online dokumentációt).
A PeaZip egy kereszt-platformos alkalmazás, futtatható 32 és 64 bites Windows (9x, 2000, XP, Vista) és Linux rendszereken.

Letöltések és funkció változások: http://peazip.sourceforge.net/
Letöltések: http://sourceforge.net/projects/peazip/
Online súgó: http://peazip.sourceforge.net/peazip-help.html
GYIK: http://peazip.sourceforge.net/faq.html
