アーカイブを作成

1) システムのコンテキストの「追加」からドラッグアンドドロップかアプリケーションのメニューボタンでアーカイブレイアウトを追加してください。
2) アーカイブを作成するために「作成」ボタンをクリックしてください (そうでなければ、次回以降のためにアーカイブレイアウトを保存できます)。

使用:
- オプションタブではアーカイブの形式･圧縮･暗号化のパラメータを設定
- I/Oタブでは入力/出力の設定
- フィルタータブでは、形式が可能であれば高度な除外/含めるのフィルタ設定
- コンソールタブではジョブの構文をカスタマイズ･分析、またはコマンドラインとしてジョブをエクスポート
代わりに、システムのコンテキストメニューから、アーカイブに置くそれぞれの項目を一度に追加できます。またSendToメニューから複数の項目アーカイブに追加できます。

アーカイブのレイアウトはさらに:
- オブジェクトをダブルクリックして開く
- ファイルを選択し、"\" キーを押すか、ファイルツール > ファイル分割 を選択し分割
- ファイルを選択し、"<" キーを押すか、ファイルツール > ファイル比較 を選択し他のファイルとバイト単位で比較
- 複数のファイルを選択し、"?" キーを押すか、ファイルツール > ファイルのチェックサム/ハッシュ を選択しファイルをチェックサム/ハッシュする
- 複数のファイルを選択し、"*" キーを押すか、セキュアな消去を 選択し安全な消去


アーカイブを 開く/アーカイブ

1) システムのコンテキストからアーカイブを開き、メニューのエントリーまたはファイルの関連付けによって移動させる、またはもう一つかPeaZipブラウザのアーカイブを選択してください。
2a) アーカイブを解凍する「全て解凍」か、 「全てここに解凍」をクリックし、出力パスを指定してください。
2b) ダブルクリックするか、Enterキーを押し、アーカイブオブジェクトを関連付けられたアプリケーションでプレビューするか、アーカイブであればPeaZipで開くか、ディレクトリであればエクスプローラーで開いてください。
2c) アーカイブオブジェクトを選択し、ホイールボタンを押すか、スペースキーを押し、出力パスを指定し解凍してください。
2d) PeaZipブラウザで右クリックし、選択した解凍や表示中のオブジェクト、プレビュー、テスト、 ファイルを追加･削除する などを実行するにはコンテキストメニューを開いてください。
ヒント: 新しいディレクトリに解凍する場合、アーカイブブラウザでコンテキストメニューを開く、または I/O tab 、"\"かCtrl+Nを押し、「新規フォルダに解凍」に設定してください

使用:
- オプションタブでは形式の特定のパラメータを設定
- I/Oタブでは入力/出力の設定、「新規フォルダに解凍」に設定(コンテキストメニューでも可能);
- フィルタータブでは、形式が可能であれば高度な除外/含めるのフィルタ設定
- コンソールタブではジョブの構文をカスタマイズ･分析、またはコマンドラインとしてジョブをエクスポート

システムのコンテキストメニューかPeaZipブラウザから複数のアーカイブを一度に解凍できます。

オプションタブの「カスタム」形式を選択することで、実行可能な標準のPeaZipで使われないアーカイブ形式をサポートできます(作成･解凍のため)

PeaZipについて

PeaZip is an open source file and archive manager: cross platform, available as portable and installable software for 32 and 64 bit Windows (9x, 2k, XP, Vista) and Linux (PeaZip is a desktop neutral application).

PeaZip allows to apply powerful multiple search filters to archive's content; create and extract multiple archives at once; create self-extracting archives; export job definition as command line; save archive's layouts; bookmark archives and folders; scan and open with custom applications compressed and uncompressed files etc...

Other features: strong encryption, robust file copy, split/join files (file span), secure data deletion, compare, checksum and hash files, system benchmark, generate random passwords and keyfiles.

Downloads and changelog: http://peazip.sourceforge.net/
All packages: http://sourceforge.net/projects/peazip/

Support: http://peazip.sourceforge.net/peazip-help.html

Features and full list of supported formats: http://peazip.sourceforge.net/peazip-free-archiver.html 
More hints and notes for users: http://peazip.sourceforge.net/peazip-more.html

Many thanks to all people contributing to PeaZip project's growth developing Open Source software components used in this application (see website for complete list), and to all users contributing with feedback, experience and advices.
