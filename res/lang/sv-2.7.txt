SKAPA ARKIV

1) Lägg till objekt till arkivets layout från systemets kontext- och Skicka till-menyer, genom att dra och släppa eller från PeaZips knappar och menyer;
2) Klicka på "Skapa"-knappen för att påbörja skapandet av ett nytt arkiv.

Använd:
- "Inställningar"-fliken för att ställa in arkivformat samt komprimerings- och krypteringsrelaterade parametrar; 
- "I/O"-fliken för inmatnings- och utmatningsinställningar;
- "Filter"-fliken för att använda avancerade uteslutningsfilter, om det är tillgängligt för det valda arkivformatet;
- "Konsoll"-fliken för att analysera och ställa in jobbets syntax, eller exportera jobbet som en kommandorad.

Alternativt, med ett enkelt handgrepp kan du lägga till varje enskilt inmatningsobjekt till ett separat arkiv från systemets kontextmeny, och från Skicka till-menyn kan du lägga till flera objekt till ett enda arkiv.

I arkivlayouten kan du även:
- Dubbelklicka på ett objekt för att öppna det;
- Markera en fil och trycka "\" eller "Högerklicka el. Arkiv > Filverktyg > Klyv fil" för att klyva den;
- Markera en fil och trycka "<" eller "Högerklicka el. Arkiv > Filverktyg > Jämför filer" för att byte för byte jämföra den med en annan fil;
- Markera flera objekt och trycka "?" eller "Högerklicka el. Arkiv > Filverktyg > Kontrollsumme-/hashfil(er)" för kontrollsumme- och hashfiler;
- Markera flera objekt och trycka "*" eller "(Arkiv > Filverktyg >) el. högerklicka > Säker borttagning (filer)" för säker filborttagning.


ÖPPNA ARKIV

1) Öppna arkiv genom filassociation, eller från systemets kontextmeny- och Skicka till-menyval, eller från PeaZips knappar och menyer
2a) Klicka "Packa upp alla" för att packa upp arkivet, eller "Packa upp alla till" för att få välja målsökväg;
2b) Dubbelklicka eller tryck Enter på ett arkiverat objekt för att förhandsgranska det med associerat program, eller öppna det i PeaZip om det är ett arkiv, eller utforska det om det är en mapp/katalog;
2c) Markera arkiverade objekt och mittenklicka eller tryck mellanslag för att packa upp dem (och tillfrågas om målsökväg);
2d) Högerklicka för att använda kontextmenyvalen och utföra alla tillgängliga funktioner, som att packa upp markerade eller visade objekt, förhandsgranska, testa, lista, lägga till eller ta bort filer etc...
Tips: Om du vill packa upp arkivet till en helt ny mapp, använd "Packa upp till ny mapp"-väljaren (ON/OFF = PÅ/AV) i kontextmenyn (eller under "I/O"-fliken), eller tryck \ eller Ctrl+N när du är i arkivbläddraren.

Använd:
- "Inställningar"-fliken för att ställa in formatspecifika parametrar;
- "I/O"-fliken för inmatnings- och utmatningsinställningar, inklusive "Packa upp till ny mapp"-väljaren (även tillgänglig i kontextmenyn);
- "Filter"-fliken för att använda avancerade inkluderings- och uteslutningsfilter, om det är tillgängligt för arkivformatet;
- "Konsoll"-fliken för att analysera och ställa in jobbets syntax, eller exportera jobbet som en kommandorad.

Alternativt kan du packa upp flera arkiv samtidigt från systemets kontextmeny.


OM PEAZIP

PeaZip free archiver är ett portabelt, Open Source-baserat (sv. öppen källkod) arkiverings-, krypterings- och filklyvningsverktyg; som stöder och hanterar ett stort urval arkiv- och komprimeringsformat.

Skriver: 7Z, 7Z-sfx, ARC, BZ2, GZ, PAQ/LPAQ, PEA, QUAD, klyning, TAR, UPX, ZIP

Läser: Se hela listan på http://peazip.sourceforge.net/peazip-free-archiver.html 

Andra/anpassade arkivtyper kan hanteras för skrivning och läsning genom att specificera anpassade exekverbara filer och syntax under "Inställningar"-fliken.

Andra funktioner: klyva/sammanfoga filer genom filspanning (file spanning), radera filer (säker filborttagning), jämförelser, kontrollsumme- och hashfiler, systemprestandakontroll (benchmarking), skapa slumpade lösenord och nyckelfiler.

I PeaZip kan du redigera, spara and återställa arkivens layouter; applicera kraftfulla, multipla filter på arkivens innehåll; hantera flera arkiv samtidigt; exportera jobbdefinitioner som kommandorader; kryptera med AES256 etc.
PeaZip_portable är ett självständigt program med samma funktioner som PeaZip; det behöver inte installeras och kan användas från flyttbara enheter (USB-minnen, fjärrkontrollerade sökvägar etc.) utan att påverka eller förändra värdsystemet.
De vanligast använda funktionerna är integrerade i kontext- och Skicka till-menyerna (på Windows) och som .desktop-filer för kontextuella menyer (på Linux); systemintegrationen utförs automatiskt med installationspaketen och kan anpassas av användare både när det gäller installationspaket ochd portabla paket (se dokumentationen på Internet, på engelska).
PeaZip kan användas av flera plattformar och kan köras på 32- och 64-bitars Windows (9x, 2000, XP, Vista) och Linux.

Nerladdningsbara filer och versions- och ändringsloggar: http://peazip.sourceforge.net/
Alla nerladdningsbara filer: http://sourceforge.net/projects/peazip/
Hjälp (på engelska): http://peazip.sourceforge.net/peazip-help.html
Vanliga frågor (FAQ): http://peazip.sourceforge.net/faq.html
